# Ciberpank

Ниже — пример подробного технического описания изображений как постановки задачи для разработки приложения «Интерактивный лист персонажа Cyberpunk 2020». Текст сформулирован так, чтобы его можно было использовать как основу для ТЗ, user stories или проектной документации.

---

# Описание макета: интерактивный лист персонажа Cyberpunk 2020

## Общая информация

На изображениях представлен классический бумажный лист персонажа настольной RPG Cyberpunk 2020.
Лист выполнен в черно-белом стиле с характерной типографикой и визуальной эстетикой киберпанка начала 90-х.

Интерактивное приложение должно воспроизводить структуру бумажного листа, сохраняя:

* визуальное деление на секции;
* RPG-логику характеристик;
* возможность редактирования;
* автоматические вычисления;
* хранение данных персонажа;
* поддержку печати и экспорта.

* Solo

---

# 2. Основные характеристики (Stats)

Раздел содержит базовые атрибуты персонажа.

## Характеристики:

* INT — интеллект
* REF — рефлексы
* TECH — техника
* LUCK — удача
* ATTR — привлекательность
* MA — скорость
* BODY — телосложение
* EMP — эмпатия

## Требования

Каждая характеристика должна:

* хранить числовое значение;
* иметь минимальные/максимальные ограничения;
* поддерживать модификаторы;
* влиять на производные параметры.

---

# 3. Производные параметры

На листе присутствуют:

* Хладнокровие
* Бег
* Прыжок
* Подъем
* Вес
* Тип тела

## Логика

Некоторые параметры должны вычисляться автоматически:

Примеры:

* Бег = MA × коэффициент
* Прыжок = функция от MA
* Подъем = функция BODY
* Тип тела = зависит от BODY
