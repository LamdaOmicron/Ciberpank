# Cyberpunk Character Sheet - Рефакторинг

## Обзор изменений

Оригинальный файл `cyberpunk_sheet 14.py` (3460 строк) был реорганизован в модульную структуру с четким разделением ответственности.

## Новая структура проекта

```
cyberpunk_app/
├── __init__.py              # Документация и метаданные пакета
├── main.py                  # Точка входа приложения
├── core/                    # Основной модуль
│   ├── __init__.py
│   ├── main_window.py       # Главное окно приложения
│   └── tabs/                # Вкладки интерфейса
│       ├── __init__.py
│       ├── base_tab.py      # Базовый класс для вкладок
│       ├── main_tab.py      # Основная информация
│       ├── skills_tab.py    # Навыки
│       ├── cybernetics_tab.py # Кибернетика
│       ├── weapons_tab.py   # Оружие
│       ├── equipment_tab.py # Экипировка
│       ├── lifepath_tab.py  # Lifepath
│       └── cyberdeck_tab.py # Cyberdeck
├── styles/                  # Стили и темы
│   ├── __init__.py
│   ├── themes.py            # Менеджер тем
│   └── styles.py            # Провайдер стилей
├── widgets/                 # Кастомные виджеты
│   ├── __init__.py
│   ├── skill_type_widget.py
│   ├── icon_combo_box.py
│   ├── flip_widget.py
│   └── combo_box_delegate.py
├── utils/                   # Утилиты
│   ├── __init__.py
│   ├── validators.py        # Декораторы валидации
│   └── logger.py            # Настройка логирования
└── data/                    # Модели данных (будущее расширение)
```

## Ключевые улучшения

### 1. Модульность
- Разделение UI компонентов на отдельные классы
- Выделение стилей в отдельный модуль `StyleProvider`
- Изоляция тем оформления в `ThemeManager`

### 2. Читаемость кода
- Типизация параметров и возвращаемых значений
- Docstrings для всех классов и методов
- Единый стиль именования (snake_case для функций/методов, PascalCase для классов)

### 3. Расширяемость
- Базовый класс `BaseTab` для всех вкладок
- Миксин `LoggerMixin` для логирования
- Конфигурируемый делегат `CustomComboBoxDelegate`

### 4. Поддерживаемость
- Логирование через стандартный logging
- Централизованное управление темами
- Переиспользуемые стили через `StyleProvider`

## Запуск приложения

```bash
python -m cyberpunk_app.main
# или
python cyberpunk_app/main.py
```
